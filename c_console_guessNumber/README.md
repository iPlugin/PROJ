[ Title: Guess the Number ]

Functional:
- Random number generation
- Checking the numbers entered by the user.
- Providing hints

===========================================================

[ For start, you need: ]

*For compiling software code*
- $ g++ main.c -o main.o
*For start the program*
- $ ./main.o
*For delete files created during compilation*
- $ rm main.o
